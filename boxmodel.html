CSS Box Model

The CSS Box Model defines how elements are sized, positioned, and rendered on a webpage. When a browser loads an HTML document, it creates a DOM tree and assigns a box to each element. This box calculates the element's dimensions and position relative to its parent or the root element, ensuring accurate layout and spacing.
Content: The area where text or other content is displayed.
Padding: Space between the content and the element's border.
Border: A frame that wraps around the padding and content.
Margin: Space between the element's border and neighbouring elements.

Padding

Padding is the space between the content of an element and its border. It creates breathing room around the content, enhancing readability and visual appeal. Padding can be set uniformly for all sides or individually for each side (top, right, bottom, left) using CSS properties like padding, padding-top, padding-right, padding-bottom, and padding-left. ex:- padding: 20px 10px 15px 5px; = padding: top right bottom left
padding: 20px; = padding to all sides
padding: 10px 20px; = padding: top-bottom right-left

Margin

Margin is the space outside the border of an element that separates it from other elements. It helps create distance between elements, preventing them from appearing too close together. Margins can be set uniformly for all sides or individually for each side (top, right, bottom, left) using CSS properties like margin, margin-top, margin-right, margin-bottom, and margin-left. ex:- margin: 20px 10px 15px 5px; = margin: top right bottom left
margin: 20px; = margin to all sides
margin: 10px 20px; = margin: top-bottom right-left
